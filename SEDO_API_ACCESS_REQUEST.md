# Email Template: Requesting Sedo API Access

## Recommended Contact Method

**Option 1**: Through Sedo Partner Program Registration
- Visit: https://sedo.com/member/partner/integration.php
- Fill out the integration request form

**Option 2**: Direct Email
- Check Sedo website for current support contact
- Common emails: support@sedo.com, partners@sedo.com, api@sedo.com

---

## Email Template

### Subject Line Options:
- `API Access Request - Parking Statistics Integration`
- `Request for Sedo API Credentials - Partner Program`
- `Integration Request: Sedo API Access for Reporting Dashboard`

---

```

**Company Details:**
- Company Name: EKBHR MEDIA
- Website: ekbhr.com
- Email: [your-email@ekbhr.com]
- Contact Person: [Your Name]

**Purpose of API Access:**
We need API access to retrieve parking statistics including:
- Daily revenue data
- Clicks and impressions metrics
- Geographic breakdown of traffic
- Device type breakdowns
- Historical reporting data

This integration will allow us to:
- Build a unified reporting dashboard
- Automate data collection and analysis
- Provide better insights to our team
- Monitor performance across our domain portfolio

**Integration Details:**
- We are using the Sedo API to integrate parking statistics into our internal reporting system
- The integration will be used exclusively for our own domain portfolio management
- We comply with all Sedo terms and conditions

**Requested Access:**
We are specifically interested in accessing the following API methods:
- DomainParkingFinalStatistics - For retrieving parking statistics
- Any other relevant methods for portfolio management

**Next Steps:**
Please let us know:
1. The approval process for API access
2. Required documentation or information
3. Timeline for credential provisioning
4. Any fees or requirements associated with API access

We have reviewed the Sedo Partner Program information at https://sedo.com/member/partner/integration.php and are ready to proceed with the integration request process.

Thank you for your time and consideration. We look forward to working with Sedo and integrating your API into our reporting infrastructure.

Best regards,

[Your Name]
[Your Title]
EKBHR MEDIA
[Your Email]
[Your Phone Number]
[Date]
```

## Key Points to Include

1. **Clear Purpose**: Mention you need parking statistics for reporting
2. **Company Details**: Include your company name and contact info
3. **Use Case**: Explain why you need API access
4. **Specific Request**: Mention the methods you need (DomainParkingFinalStatistics)
5. **Professional Tone**: Keep it formal and business-like
6. **Next Steps**: Ask about the process and timeline

## After Sending

1. **Wait for Response**: Sedo may take several business days to respond
2. **Follow Up**: If no response in 1 week, send a polite follow-up
3. **Documentation**: Once approved, request API documentation if not provided
4. **Credentials**: They will provide SignKey and Partner ID
5. **Update Environment**: Add credentials to `.env.local` once received

---

## Tips

- **Be Professional**: Use formal business language
- **Be Specific**: Clearly state what you need
- **Be Patient**: Approval processes can take time
- **Be Ready**: Have your integration details prepared
- **Be Compliant**: Mention you'll comply with their terms
